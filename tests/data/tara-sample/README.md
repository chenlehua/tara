# TARA 分析样例数据

本目录包含用于测试一键生成 TARA 报告功能的示例数据。

## 文件说明

| 文件 | 描述 |
|------|------|
| `assets.json` | JSON 格式的资产清单，包含8个典型汽车电子电气资产 |
| `assets.csv` | CSV 格式的资产清单，可用于批量导入 |
| `architecture.svg` | 智能网联汽车电子电气架构图 |

## 资产列表

1. **整车控制器 (VCU)** - ECU 类型，CAN/Ethernet 接口
2. **电池管理系统 (BMS)** - ECU 类型，CAN/LIN 接口
3. **智能网关 (CGW)** - Gateway 类型，多接口
4. **远程通信单元 (T-Box)** - Telematics 类型，无线接口
5. **信息娱乐系统 (IVI)** - ECU 类型，多媒体接口
6. **自动驾驶域控制器 (ADAS DCU)** - 域控制器
7. **OBD诊断接口** - 诊断接口
8. **充电通信控制器 (SECC)** - 充电接口

## 使用方法

在前端"一键生成报告"页面上传这些文件即可开始 TARA 分析：

1. 上传 `assets.json` 或 `assets.csv`
2. 上传 `architecture.svg` (可选)
3. 选择报告模板
4. 点击"开始分析"

## 注意事项

- 文件编码均为 UTF-8
- CSV 文件使用逗号作为分隔符
- SVG 文件符合标准 SVG 1.1 规范
